dog retinal-front-end report

Scope note: This page traces the modeled retinal front end from scene assumptions to retinally delivered irradiance, receptor sampling, and retinal-information renderings. It is not a direct perceptual, retinal-circuit, or cortical report.

Scene and patch

Scene input mode: reflectance_under_d65

Spectrum source: RGB-inferred

Patch geometry: full scene 14.25 x 10.35 deg, simulated patch 2.00 x 1.45 deg.

  • RGB is interpreted as a display-referred color that is reconstructed into a Smits-inspired D65-optimized reflectance spectrum.
  • The recovered spectrum is inferred from RGB and is not a unique physical scene spectrum.
  • This mode models diffuse reflectance under D65 for comparative PoC work.
